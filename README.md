# Personal Portfolio Website

## Overview

A modern and responsive personal portfolio website developed using HTML, CSS, and JavaScript. The portfolio showcases my skills, projects, internships, certifications, and professional information in an interactive and visually appealing manner.

The website is designed to provide visitors with a comprehensive overview of my technical background, achievements, and career aspirations.

## ✨ Features

- Responsive and modern user interface
- Light and Dark theme support
- Language switching functionality
- Interactive hero section
- Skills showcase
- Projects portfolio section
- Internship and experience timeline
- Certifications display
- Interests section
- Contact information section
- Smooth scrolling and animations
- Mobile-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
